# Step0. Requirements
 - Installation USB
 - Samsung Laptop

# Step1. Bios 진입
<kbd>F2</kbd> 


# Step2. Boot 설정
 - Secure Boot Control <kbd>off</kbd>
 - Fast BIOS Mode <kbd>off</kbd>
 - OS Mode Sslection <kbd>CSM and UEFI OS</kbd>
 - <kbd>CSM and UEFI OS</kbd>
 - Boot Device Priority <kbd>USB</kbd> 

# Step3. 재실행
 - <kbd>SAVE</kbd>
